## 프로젝트 소개
YOU-MAP은 서강대학교 학생들중 학교길, 꿀팁 그리고 장소 추천등 다양한 학교 이용 방법 추천을 제공하는 서비스입니다.

본인의 전공의 건물뿐만 아니라 타 건물들의 용도와 꿀팁까지!!!

커뮤니티 창을 통하여 여러가지 학교 생활을 꿀팁을 공유해 보아요!

만족스러운 학교생활을 위하여 유저가 되어보세요!
